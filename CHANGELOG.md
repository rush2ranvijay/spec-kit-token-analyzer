All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-01

### Added

- `speckit.token-analyzer.baseline` — Capture baseline token metrics for a test scenario
- `speckit.token-analyzer.compare` — Compare token consumption between two runs
- `speckit.token-analyzer.report` — Generate token analysis report from captured data
- Bash and PowerShell scripts: `capture-tokens`, `estimate-tokens`, `generate-report`, `score-quality`
- Report template for comparison output
- Quality scoring with weighted five-factor formula
